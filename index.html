How to get the HTML source
  1. Open your Copilot share link in Chrome or Edge and wait for it to fully load
  2. Press Ctrl+U (Windows) or ⌘+U (Mac) to open View Source in a new tab
  3. Press Ctrl+A to select all, then Ctrl+C to copy
  4. Paste it into the box below and click Convert

How to save the HTML file
  1. Open your Copilot share link in Chrome or Edge and wait for it to fully load
  2. Press Ctrl+S (Windows) or ⌘+S (Mac) to save the page
  3. In the save dialog, set the format to Webpage, Single File (*.mhtml) or Webpage, Complete (*.html)
  4. Upload that file below

How to get the plain text
  1. Open your Copilot share link in a browser and wait for the conversation to fully load
  2. Click anywhere on the page, then press Ctrl+A to select all visible text
