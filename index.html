Copilot Chat → Markdown

Extract the full text of a Microsoft Copilot shared conversation and export it as a clean, structured Markdown file.
Get one at aistudio.google.com
Fetch from shared URL
  1. Paste a public Copilot share URL (https://copilot.microsoft.com/shares/...)
  2. Click Convert and the app will try to fetch page content automatically
  3. If browser fetch is blocked, use node scripts/save-copilot-share-html.js ... then Upload HTML File

Tip: the Fetch URL → Download .html button may fall back to proxy text in restricted browsers. For highest-fidelity saved HTML, use the CLI helper script.


One-time setup
  1. Show your bookmarks bar: Ctrl+Shift+B
  2. Drag the blue button above to the bar
  3. Done — you only do this once
Every conversation
  1. Open your Copilot share URL and wait for it to load
  2. Click Export Chat in your bookmarks bar
  3. copilot-chat-[date].md downloads instantly ✓

Want to automate this? Run node scripts/save-copilot-share-html.js https://copilot.microsoft.com/shares/... and upload the generated file.

Why a bookmarklet and not a URL field?
Copilot share pages are rendered entirely by JavaScript — no external tool can fetch their content from a URL. The bookmarklet runs inside your browser tab where the page is already live, extracts the conversation text, formats it as Markdown, and downloads the file. No server, no API key, no intermediate steps.

How to get the HTML source
  1. Open your Copilot share link and wait for it to fully load
  2. Press Ctrl+U to open View Source in a new tab
  3. Press Ctrl+A then Ctrl+C to copy all
  4. Paste below and click Convert
How to get the plain text
  1. Open your Copilot share link and wait for it to fully load
  2. Press Ctrl+A then Ctrl+C to copy all visible text
  3. Paste below and click Convert
